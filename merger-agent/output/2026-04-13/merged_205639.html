Claude Mythos Withheld From Public Release Over Cybersecurity Risks; Limited to 50 Organizations via Project Glasswing

📅 April 13, 2026

Anthropic announced it will not publicly release Claude Mythos, its most powerful model to date, citing its ability to autonomously discover and exploit zero-day vulnerabilities in every major OS and web browser. The model found thousands of previously unknown vulnerabilities, including a 27-year-old flaw in OpenBSD and chained Linux kernel vulnerabilities to escalate privileges. Access is restricted to roughly 50 vetted organizations including Google, Microsoft, Amazon, and IBM through Project Glasswing. Internal testing revealed Mythos could help non-experts exploit vulnerabilities and autonomously perform sophisticated cyberattacks.

Anthropic הודיעה שלא תשחרר לציבור את Claude Mythos, המודל החזק ביותר שלה עד כה, בטענה שהוא מסוגל לגלות ולנצל באופן אוטונומי פרצות zero-day בכל מערכת הפעלה ודפדפן מרכזיים. המודל מצא אלפי פרצות שלא היו ידועות בעבר, כולל פרצה בת 27 שנה ב-OpenBSD ושילוב של פרצות Linux kernel להגדלת הרשאות. הגישה מוגבלת לכ-50 ארגונים מוכחים כמו Google, Microsoft, Amazon ו-IBM דרך Project Glasswing. בדיקות פנימיות חשפו שהמודל יכול לעזור לחסרי ניסיון לנצל פרצות ולבצע התקפות סייבר מתוחכמות.
