📅 April 4, 2026

Anthropic ended the ability for Claude Pro and Max subscribers to route usage through third-party frameworks like OpenClaw, effective April 4, 2026, citing 'outsized strain' on systems not designed for such usage patterns. Developers who continue using those tools with Claude must now pay separately under a new 'extra usage' billing tier, with API costs potentially exceeding $1,000/month versus $200/month for the Max plan — a policy set to extend to other third-party harnesses soon. Separately, Anthropic accidentally exposed Claude's source code by including a source map file in its 2.1.88 npm package release, leading to thousands of copies being uploaded to GitHub before Anthropic initiated DMCA takedowns — drawing sharp criticism given ongoing debates about AI training data sourcing. The dual crises have triggered significant backlash from the open-source developer community, with the HN thread drawing 1,077 upvotes and 818 comments.

Anthropic הפסיקה את היכולת של מנויי Claude Pro ו-Max להנתיב שימוש דרך frameworks של צד שלישי כמו OpenClaw, החל מ-4 באפריל 2026, בטענה של 'עומס יתר' על מערכות שלא תוכננו לדפוסי שימוש כאלה. מפתחים שימשיכו להשתמש בכלים אלה עם Claude חייבים כעת לשלם בנפרד תחת רמת חיוב חדשה של 'שימוש נוסף', כאשר עלויות API עלולות להיות יותר מ-$1,000 בחודש לעומת $200 בחודש עבור תוכנית Max — מדיניות שנקבעה להרחיב לרתמות צד שלישי אחרות בקרוב. בנפרד מכך, Anthropic חשפה בטעות את קוד המקור של Claude על ידי הכללת קובץ source map בהוצאת חבילת npm 2.1.88 שלה, מה שהוביל להעלאה של אלפים של עותקים ל-GitHub לפני ש-Anthropic יזמה הסרות DMCA — מה שהביא לביקורת חדה בהתחשב לדיונים מתמשכים על מקורות נתוני אימון של AI. משברים כפולים אלה הנעירו תגובה משמעותית מקהילת המפתחים בקוד פתוח, כאשר הפתיל ב-HN זכה ל-1,077 upvotes ו-818 תגובות.

📅 April 4, 2026

Google DeepMind released Gemma 4, a family of four open-weight multimodal models licensed under Apache 2.0, designed to run on a single 80GB Nvidia H100 GPU and spanning edge devices to data center deployments. The flagship 31B-parameter model scored 89.2% on the AIME 2026 math benchmark, compared to 20.8% for the prior Gemma 3 27B model, while a smaller E4B edge model outperforms Gemma 3 27B on most benchmarks at roughly one-sixth the size. The release includes Gemma's first Mixture-of-Experts (MoE) model, supports image, text, and audio inputs across over 140 languages, and is optimized for agentic workflows — building on an ecosystem of 400M+ prior downloads and over 100K community-created Gemmaverse variants. NVIDIA collaborated on hardware optimizations and the models are available on Hugging Face with Apache 2.0 licensing, enabling fully open commercial deployment.

Google DeepMind שחררה את Gemma 4, משפחה של ארבעה מודלים מולטימודליים בעלי משקל פתוח תחת רישיון Apache 2.0, שתוכננו לפעול על GPU יחיד של Nvidia H100 בגודל 80GB וגם על התקנים edge וגם על פריסות datacenter. מודל ה-flagship בעל 31B פרמטרים הביא ניקוד של 89.2% בבנצ'מארק המתמטיקה AIME 2026, בהשוואה ל-20.8% למודל Gemma 3 בעל 27B קודם, בעוד שמודל edge E4B קטן יותר עולה על Gemma 3 27B ברוב הבנצ'מארקים בגודל של כשישית בערך. ההוצאה כוללת את מודל Mixture-of-Experts (MoE) הראשון של Gemma, תומכת בתשומות של תמונה, טקסט וקול על פני למעלה מ-140 שפות, ומותאמת לזרימות עבודה agentic — בהמשך לאקוסיסטם של 400M+ הורדות קודמות וערכים של Gemmaverse שנוצרו על ידי קהילה מעל 100K. NVIDIA שיתפה פעולה על אופטימיזציות חומרה והמודלים זמינים ב-Hugging Face עם רישיון Apache 2.0, המאפשר פריסה מסחרית פתוחה לחלוטין.
