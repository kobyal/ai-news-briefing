OpenAI released GPT-5.5 (codenamed 'Spud'), its most capable model to date and the first fully retrained base model since GPT-4.5. OpenAI השיקה את GPT-5.5, בשם הקוד 'Spud' — המודל החזק ביותר שלה עד כה, והראשון שעבר אימון מחדש מלא מאז GPT-4.5. המודל תוכנן לביצוע משימות מחשב מורכבות ורב-שלביות עם מינימום הכוונה אנושית, והשיג 82.7% על Terminal-Bench 2.0 ו-84.9% על GDPval. מבחינת latency לכל token הוא שווה ל-GPT-5.4, אך דוחף את הגבול קדימה בתחומי agentic coding, שימוש במחשב ועבודת ידע. גישת API נדחית בינתיים לצורך בדיקת בטיחות נוספת.

Alibaba's Qwen Team released Qwen3.6-27B, the first dense open-weight model in the Qwen 3.6 family.
